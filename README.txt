Enable Heatmaps, Recordings and funnels for free in your webiste.

== Description ==

Waypanel.com is a free software that allows you to have great information about your website customers.

Create instant heatmaps, Record every user in your wesbite. Create smart funnels to help you to understand the flaws in your convertions and also create alerts to tell you if anything is wrong with your website!

Waypanel is very easy to install and start receiving data!


== Installation ==

1. Upload `waypanel-heatmap` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Create your account in waypanel.com
4. Place the tracking script in the Settings area of the plugin in your wordpress
5. You are good to go! Have fun!
